## V2.1 (2026-07-10)

### V2.1.0-RC1 (2026-07-10)
- **核心闭环执行层建设**：新增 `/画像`, `/修改`, `/诊断` 工作流。教师画像字段结构化增强，强制基于明确提供的证据提取，默认不跨课程迁移。修改流程要求产生基于事实回滚风险的最小 Diff。诊断流程和规划流程的职责进一步厘清。
- **适配器弹性化与局部降级**：所有适配器坚守绝不捏造、绝不越权、绝不侵犯隐私的硬底线。对于诸如要求源码视频、原始数据等特定表现形式动作，全面降级为“满足规定与隐私且有收益时建议提供”。
- **统计学决策树**：更新了实证论文的适配器，废除了单一变量类型二分法，植入包含 8 步分析与条件检验（如 Fisher 检验）的统计决策树。强制封杀从普通相关性分析推演干预阈值与因果关系的违规动作。并对示例库中对应的违规内容一并进行了清除。
- **独立大模型语义评测靶场 (`evals/`)**：建立独立评测框架体系，包含基于事实真实性验证、统计伦理、画像约束等 16 个高难度测试对抗案例。用于在接入大模型 API 后的端到端验证，不再阻塞无网络的常规单元测试。
- **发布状态与门禁优化**：统一全仓说明中版本与 GitHub Actions 徽章，补强 BOM UTF-8 检测漏洞，更新 Bundle 打包序列及 Manifest 断言规则，确保 Bundle 随工作流无缝更新。
