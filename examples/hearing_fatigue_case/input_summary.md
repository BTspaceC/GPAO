# 案例：大学生耳机使用习惯与听觉疲劳相关性分析

## 1. 任务背景
- **课程**：社会科学统计基础
- **作业要求**：设计问卷，收集不少于30份数据，使用统计软件进行分析，提交不少于3000字的调查报告。
- **评分标准**：
  - 问卷设计与数据质量 (25%)
  - 统计分析深度 (35%)
  - 报告结构与图表规范 (25%)
  - 结论建议 (15%)

## 2. 原始状态（得 80 分的版本）
- **数据**：通过问卷星收集了35份同学的数据。并在 Excel 中手动删除了3份全部选C的废卷。
- **分析工具**：SPSS。
- **正文内容**：
  - 描述了男女比例。
  - 画了几个饼图，显示“每天戴耳机超过3小时的人占60%”。
  - 结论：“大学生耳机使用时间过长，容易导致听觉疲劳，建议减少使用时间。”
- **附录**：
  - 附带了一大堆 SPSS 跑出来的冗长表格截图（几十页），包含了频数表、描述性统计等，但没有在正文中引用它们。

## 3. 老师评语
“问卷设计基本合理，但数据分析仅停留在表面。没有深入挖掘变量之间的相关关系。附录截图过于冗长且无意义。”
