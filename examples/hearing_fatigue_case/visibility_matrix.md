# 评分点可见性矩阵 (示例)

`schema_version: 1`

**声明：本示例引用自 `case_facts.md`，状态为 [VERIFIED] 的事实才能作为评价基准。**

| 评分项目 | 满分 | 目标可见形式 | 现状可见形式 | 差距分析 | 改进方案优先级 |
| :--- | :---: | :--- | :--- | :--- | :---: |
| 1. 数据收集质量 | [PLACEHOLDER] | 清洗前后的样本量对比表格，清洗逻辑说明。 | 仅在摘要中一句话：“收集了32份问卷”。 | 老师看不到数据处理的过程，容易被认为数据不可靠。 | P0 (严重影响真实性感知) |
| 2. 统计分析准确性 | [PLACEHOLDER] | 三线表展示相关系数矩阵，配以置信区间或P值星号。 | 正文提到“有相关性”，没有图表和统计值。 | 结论像是个人主观感受，缺乏统计学支撑。 | P1 (核心分值) |
| 3. 图表与学术规范 | [PLACEHOLDER] | 统一格式的高质量矢量图，明确的表头和图注。 | 截图粘贴的 SPSS 默认图表，缺乏编号。 | 不符合老师对规范性的明确要求 (P_TCH_01)。 | P1 (明确事实支持) |
| 4. 结论与建议深度 | [PLACEHOLDER] | 结合统计结果提出的具体建议，讨论研究局限性。 | 提出了泛泛的建议。 | 缺乏具体数据的直接支撑，显得空洞。 | P2 (拉开分差) |
