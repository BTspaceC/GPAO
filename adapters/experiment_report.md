# 适配器：实验报告 (Experiment Report)

## 适用范围
物理、化学、生物、工程材料等理工科课程，强调现象观察、数据记录和误差分析。

## 核心评分点转换表

| 常见评分点 | 在 GPAO 中的等效要求 | 报告/正文应体现的位置 |
| :--- | :--- | :--- |
| **实验原理与目的 ** | 必须准确写出依据的公式或定理。 | 报告开头 |
| **实验仪器与步骤 ** | 不要照抄指导书，要用自己的话简述实际操作流程，如有改动必须注明。 | "实验操作"小节 |
| **原始数据记录 ** | 数据必须真实，禁止伪造。要有清晰的数据表格和带单位的坐标图。 | "数据记录与处理"小节 |
| **数据处理与结果分析 ** | 详细展示从原始数据到最终结果的计算步骤。 | "数据分析"小节 |
| **误差分析与讨论 ** | 是拉开分差的关键。不能只写“存在系统误差”，必须具体到哪一步操作、哪个仪器的什么特性导致了什么方向的误差。 | "误差分析与讨论"小节 |

## 高辨识度亮点设计池 (P2 优先级)

1. **详细的误差量化传播分析**：不仅列出误差来源，还运用误差传递公式，计算每个误差源对最终结果的影响权重。
2. **实验异常现象记录**：真实记录某次失败的尝试或突发状况，并分析原因（例如："在第3次读数时发生温度骤降，导致数据作废，故重新记录该组数据"）。
3. **改进方案提出**：针对本次实验的局限性，提出具体的仪器改进或步骤优化建议。



**默认权重**：禁止提供。仅当任务书、评分表或老师明确要求中存在分值时填写。没有正式权重时，只能描述该项可能的重要性，不得生成具体百分比。

## 常见冲突处理

- **数据不完美 vs 标准答案偏好**：当实际测量数据与理论值偏差较大时，**严禁篡改数据去迎合理论值**。应该如实呈现偏离的数据，并在讨论部分花重墨进行深刻的误差溯源分析。通常，对真实错误数据的深刻反思得分，高于伪造的完美数据。

## 附件与提交要求

为了确保“证据链完整性”，必须提交：
1. **原始数据记录单**：实验课上老师签字的原始草稿扫描件或照片（最强真实性证据）。
2. **数据处理代码或表格**：如果使用了 MATLAB/Python/Excel 处理数据，附上对应文件。
*(正文中应声明："图表计算基于真实原始数据，签字版原始记录单见附件 A")*
